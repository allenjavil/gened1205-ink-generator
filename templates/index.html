Chinese Character Ink Generator

Draw Chinese characters, collect them as visual elements, then generate one ink-style artwork.

Project Concept

From Stroke to Prompt

My technology essay focused on the distance between using a writing system and fully understanding how it is constructed. I called this an interface gap: the space between knowing what you want to express and knowing how the system actually builds it. To extend on this idea, I wanted to make this gap visible. This project begins with a handwritten Chinese character. The user draws by hand, the system reads the character through OCR, connects it to a curated meaning, and transforms that meaning into an ink-style generated image.

Overall, this project seeks to explore what happens when writing moves between hand, machine, and image. The first mark is physical and personal, but the final composition is completed through automation. In that shift, the character becomes both a written form and a prompt.

How It Works

The Process

1. Draw

A character is written by hand on the canvas.

2. Recognize

OCR identifies the character from the drawing.

3. Interpret

A CSV-based dictionary connects the character to a curated meaning.

4. Generate

The collected meanings become one ink-style artwork.
